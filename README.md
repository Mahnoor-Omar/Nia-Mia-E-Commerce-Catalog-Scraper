#  E-commerce Web Scraper

A Python-based web scraper that crawls a Shopify e-commerce website and extracts product data.

## Features

- Auto-discovers collections
- Handles pagination
- Extracts:
  - Product Name
  - Price
  - Product URL
  - Collection
- Removes duplicates
- Exports CSV dataset

## Tech Stack

- Python
- Requests
- BeautifulSoup
- Pandas

## Run

```bash
python scraper.py
